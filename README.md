🏦 Banking Application System (Java)

📌 Description

The Banking Application System is a Java-based console application that simulates basic banking operations.
It allows users to manage bank accounts, perform transactions, and view account details.

This project is great for learning Java, Object-Oriented Programming (OOP), and real-world application logic.

---

🚀 Features

- 🆕 Create a new bank account
- 💰 Deposit money
- 💸 Withdraw money
- 📊 Check account balance
- 🔍 View account details
- 🔐 Basic input validation for transactions

---

🛠️ Technologies Used

- Java
- OOP Concepts (Classes, Objects, Encapsulation, Abstraction)
- ArrayList / Collections
- Scanner (for user input)

▶️ How to Run

1. Compile the program

javac Account.java
javac Bank.java
javac BankingApplication.java

2. Run the program

java BankingApplication

⚠️ Rules & Validations

- Minimum balance should be maintained
- Withdrawal amount should not exceed balance
- Account number must be unique
- Handle invalid inputs carefully
